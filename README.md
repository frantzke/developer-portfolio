## How To Use

1. Clone into your local machine.
```bash
$ git clone https://github.com/frantzke/developer-portfolio.git
``` 
2. Go into the repository
```bash
$ cd developer-portfolio
```

3. Install dependencies
```bash
$ yarn
```

4. Start's development server
```bash
$ yarn dev
```
